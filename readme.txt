== Description ==
Published content manager which creates a copy of updated content and keeps the current published copy intact. One must publish the new custom pending copy which will update the original post and you may view the updated content. This also works with the custom fields.

== Installation ==
1. Upload the plugin folder to your /wp-content/plugins/ folder
2. Install as a regular WordPress plugin.
3. Done.
